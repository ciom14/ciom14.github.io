# ciom14.github.io

Портфолио лабораторных работ по курсу «Программирование на Python».

## Тема оформления

Используется тема **Material for MkDocs**.

Выбор обоснован:
- Профессиональный внешний вид, подходящий для портфолио
- Встроенная поддержка русского языка
- Удобная иерархическая навигация с вкладками
- Широко применяется в реальной документации (FastAPI и др.)
- Переключение между светлой и тёмной темой
