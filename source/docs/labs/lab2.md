# Лабораторная работа 2

**Тема:** Основы NumPy: массивы и векторные операции

---

## 1. Цель работы и задание

Реализовать набор функций для работы с массивами, матрицами, статистического анализа и визуализации данных с использованием библиотек NumPy, Pandas, Matplotlib и Seaborn. Все функции должны:

- проходить автоматические тесты (pytest); 
- иметь аннотации типов (PEP-484);
- содержать docstring (PEP-257);
- соответствовать стилю PEP-8.

---

## 2. Выводы
 
В ходе лабораторной работы были реализованы и протестированы 17 функций, охватывающих ключевые возможности NumPy.
 
**Массивы и матрицы.** NumPy предоставляет удобные инструменты для создания и преобразования массивов (`arange`, `reshape`, `.T`). Операции выполняются без явных циклов, что делает код чище и быстрее по сравнению со стандартными списками Python.
 
**Векторные и матричные операции.** Операторы `+`, `*` и `@` работают поэлементно или как полноценное матричное умножение в зависимости от контекста. Это позволяет записывать математические выражения в естественной форме. `np.linalg` покрывает все базовые задачи линейной алгебры: определитель, обратная матрица, решение систем уравнений.
 
**Статистический анализ.** Связка `pandas` (загрузка CSV) + `numpy` (вычисления) оказалась удобной и лаконичной. Функции `mean`, `median`, `std`, `percentile` позволяют получить полную картину распределения данных буквально в несколько строк.
 
**Визуализация.** `matplotlib` и `seaborn` хорошо дополняют друг друга: первый — для базовых графиков (гистограмма, линейный), второй — для статистических (тепловая карта).
